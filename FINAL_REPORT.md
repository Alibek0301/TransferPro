```bash
# 1. Замените домены на реальные (если нужно)
# 2. Выполните:
git add .
git commit -m "SEO: Optimize for search visibility"
git push origin main

# GitHub Actions автоматически деплоит!
# Через 5 минут сайт будет live с SEO оптимизацией
```

### **Вариант 2: Собственный домен (Netlify)**
```bash
npm install -g netlify-cli && netlify deploy --prod
# Через 2 минуты деплой готов
```

### **Вариант 3: VPS сервер**
```bash

scp -r dist/* user@transferpro.kz:/var/www/
# Копируйте файлы на сервер
```
