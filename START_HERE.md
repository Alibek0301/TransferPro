
- `VITE_GA_MEASUREMENT_ID` - Google Analytics 4 ID (опционально)
- `VITE_YANDEX_METRIKA_ID` - Yandex Metrika ID (опционально)
- `VITE_FB_PIXEL_ID` - Facebook Pixel ID (опционально)
