# TransferPro Landing

## Быстрый запуск

```bash
npm install
npm run dev
```

Vite теперь запускается на `0.0.0.0:5173`, поэтому сайт доступен снаружи контейнера/VM.

## Проверка конфликтов merge

```bash
rg -n "<<<<<<<|=======|>>>>>>>" .
```

Если команда ничего не вывела — конфликтных маркеров в проекте нет.

## Если сайт не открывается

1. Убедитесь, что зависимости установились без ошибок.
2. Проверьте, что dev-сервер слушает порт 5173.
3. Для preview используйте:

```bash
npm run build
npm run preview
```

Preview слушает `0.0.0.0:4173`.
